УОС-01

Преобразователь частоты

Общие сведения

Преобразователь частоты (ПЧ) — устройство, переносящее спектр сигнала из одной частотной области в другую с сохранением закона модуляции. Состав ПЧ: смеситель, гетеродин и полосовой фильтр. На выходе выделяется сигнал промежуточной частоты uпч(t).

Процесс преобразования описывается набором комбинационных частот:

ωп = |±mωг ± nωс|, m, n = 0, 1, 2, …

Нужная составляющая (обычно ωг − ωс) выделяется полосовым фильтром. Побочные составляющие подавляются им же.

Типы смесителей

Тип 1 — Дифференциальный Аналоговое перемножение сигналов на дифференциальном каскаде. Метод «переменной крутизны». Удвоенная ЭДС, подавление гетеродина, ослабление его шумов и чётных гармоник сигнала.
Тип 2 — Двойной балансный Три дифференциальных каскада. Истинное аналоговое перемножение. При малых Uc, Uг ≪ φT выход пропорционален произведению: uвых ∝ uc(t)·uг(t).
Тип 3 — Кольцевой диодный Четыре диода в кольце. Подавляет ωc и ωг на выходе. Эффективен до нескольких МГц. При больших уровнях в каждое плечо включают несколько диодов последовательно.

Ключевые соотношения

Для дифференциального каскада при малом входном сигнале (|Uc| ≪ φT):

uвых(t) = −Rн · uc(t) · iy / 2φT

Для двойного балансного смесителя при малых сигналах:

uвых(t) ≈ (Rнi0 / 4φT²) · uc(t) · uг(t)

После перемножения гармоник на выходе:

cos(ωгt) · cos(ωсt) = ½[cos(ωг−ωс)t + cos(ωг+ωс)t]

Побочные каналы приема

Частоты помех, дающих ПЧ на выходе, определяются из:

ωпч = |±mωг ± nωп| → ωп = (m/n)ωг ± ωпч/n

Наиболее значимые каналы (fпч = 465 кГц):

    Стабильность коэффициента преобразования. Установить Uс ≈ 25–30 мВ. В 5 точках диапазона fс = 300–3000 кГц получить преобразование при fпч = 465 кГц, измерить Uпч и вычислить Кпр = Uпч/Uс для каждого типа смесителя.
  • 3
    Зеркальный канал приёма. Выбрать fг ∈ [500; 2000] кГц. Последовательно получить преобразование по основному (fг − fс = 465 кГц) и зеркальному (fс − fг = 465 кГц) каналам. Измерить Uпч для трёх смесителей при Uс = const (задание 2).

    Побочные каналы приёма. Выбрать fг ∈ [500; 1500] кГц. Измерить Uпч на частотах: fс = fг − 465, fп1 = 2fг − 465, fп2 = 2fг + 465, fп3 = 3fг − 465, fп4 = 3fг + 465 кГц. Построить спектрограммы с нормировкой к Uпч основного канала.
