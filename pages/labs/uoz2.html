Амплитудный детектор

Общие сведения

Амплитудный детектор (АД) — устройство для получения на выходе напряжения, изменяющегося по закону огибающей входного АМ-сигнала. АМ-сигнал записывается как:

uc(t) = ua(t)·cos(ωct), ua(t) = Uc[1 + max(t)], ma ≤ 1

Детектирование возможно двумя путями: нелинейное преобразование (квадратичная ВАХ) и параметрическое перемножение на опорное когерентное колебание u0(t) = U0cos(ωct). В обоих случаях после ФНЧ выделяется низкочастотный сигнал ∝ ua(t).

Основные параметры

ПараметрОпределение
Детекторная характеристика U= = f(Uc)
Крутизна SАД dU= / dUc [безразм.]
Коэффициент передачи kΩ kΩ = UΩ / (ma·Uc)
Коэф. нелинейных искажений kн √(U²2Ω+U²3Ω+…) / U1Ω
Коэффициент фильтрации kф kф = Uc / Uω = ωc·Cн·rg

Режимы работы диодного детектора

Режим «слабых» сигналов (квадратичный участок ВАХ): детекторная характеристика квадратичная, нелинейные искажения велики. Коэффициент искажений при гармонической модуляции:

kн ≈ ma / 4 → при ma = 30%: kн ≈ 7,5%

Режим «сильных» сигналов (линейный участок, диод в режиме отсечки): детектор линейный, Uвых = Uccosθ·[1 + max(t)]. Угол отсечки определяется только S и Rн, не зависит от Uc:

θ ≈ ∛(3π / (S·Rн))

Условия малых нелинейных искажений при работе на «сильных» сигналах:

RнCн·Ωmax ≤ √(1 − m²a) / ma

Типы детекторов

Тип 1 — Диодный Диод + нагрузочная RC-цепь. Заряд Cн через открытый диод, разряд через Rн. Параметры: R1 = 20 кОм, R2 = 10 кОм, C1 = 4700 пФ, C2 = 1000 пФ. ＋ Простота, надёжность − Нелинейные искажения при слабых сигналах
Тип 2 — Транзисторный Детектирование за счёт нелинейной ВАХ базового/коллекторного тока. Параметры: R1 = 20 кОм, R2 = 10 кОм, C1 = 4700 пФ, C2 = 1000 пФ. ＋ Одновременное усиление + детектирование − Сложнее в реализации, меньше входное Z
Тип 3 — Синхронный Перемножение входного АМ-сигнала на опорное колебание u0(t) = U0cos(ωct). Uс генератора: fc = 465 кГц, Uc = 0–1,0 В, ma = 30%. ＋ Минимальные нелинейные искажения, высокое Zвх − Требует точной синхронизации опорного сигнала

Синхронный детектор — влияние рассинхронизации

При фазовом сдвиге ψ ≠ 0 между опорным и несущим колебаниями амплитуда выходного сигнала:

uвых(t) = 0,5·Uc[1 + macos(Ωt)]·U0·cosψ

Максимум при ψ = 0. При ψ → π/2 сигнал пропадает. Разделительный конденсатор выбирается из условия:

Cp > 1 / (Ωmin·RУНЧ), RУНЧ ≥ 5·Rн
