УОС-04

Входная цепь

Общие сведения

Входная цепь (ВЦ) — пассивный четырёхполюсник, связывающий антенну с последующими каскадами (УРС или ПЧ). Функции ВЦ: передача энергии сигнала из антенны и предварительная фильтрация помех, в первую очередь — сигналов зеркального канала (fз = fc + 2fпч) и прямого прохождения (f = fпч).

В установке: антенна — эквивалент ненастроенной антенны с ёмкостью CA; контур перестраивается варикапом в диапазоне 250–2000 кГц. Связь с антенной коммутируется кнопками: C1, C2, C3 (ёмкостная) и M (индуктивная).

Основные характеристики

ХарактеристикаОпределение
Ku(f) Коэффициент передачи по напряжению: Uвых/EA
Ku0 Резонансный Ku — при f = f0 (максимум)
δ(f) Частотная селективность: |Ku(f)| / Ku0
ΔF0.707 Полоса пропускания по уровню 0,707 (−3 дБ)
Kд Коэффициент перекрытия диапазона: fc max / fc min

Оптимальный коэффициент включения антенны, обеспечивающий максимум Ku0 при G1 ≥ G2 + g:

p1 opt = √[(G2 + g) / G1]

При этом максимальный резонансный коэффициент передачи:

Ku0 max = 1 / [2·r'A·√(g + G2)]

На практике выбирают a = p1/p1 opt ≈ 0,5, при этом Ku0 ≈ 0,8·Ku0 max, а избирательность близка к ненагруженному контуру.

Виды связи антенны с контуром

Внешнеёмкостная (C1, C2, C3) p1 = Cсв/(CA+Cсв) ≈ Cсв/CA. Ku0 ∝ fc² — резкий рост с частотой. При Kд ≫ 1,5 неравномерность становится неприемлемой. ＋ Простота, широкое применение − Большая неравномерность Ku0 по диапазону
Индуктивная (M) p1 = M/L. Режим удлинённой антенны (fA < fc min): Ku0 ≈ const по диапазону. Оптимально: fA ≈ (0,5–0,7)·fc min. ＋ Высокое постоянство Ku0 в режиме удлинения − Нужна катушка связи, сложнее конструктивно
Комбинированная (M + C1) Сочетает ёмкостную (Ku0 растёт с частотой) и индуктивную (Ku0 убывает) связи. При правильном выборе соотношения — компенсация неравномерности, Ku0 ≈ const. ＋ Наилучшая равномерность в диапазоне − Требует точного согласования двух видов связи

Избирательность по зеркальному каналу

Подавление помехи на частоте зеркального канала fз = fc + 2fпч:

δK = √{ 1 + Q²э·[(fc+2fпч)/fc − fc/(fc+2fпч)]² }

Для увеличения δK нужно увеличивать fпч и добротность Qэ.

Вносимая расстройка антенной

При ёмкостной связи эквивалентная ёмкость антенной цепи C'A = CACсв/(CA+Cсв) подключается к контуру и расстраивает его. Допустимая вносимая ёмкость:

ΔCдоп ≤ Cэ min / Qэ

Чем больше Cсв, тем сильнее расстройка. При смене C1 → C2 → C3 расстройка нарастает; задание 5 посвящено её измерению.

Цель работы: исследование одноконтурных ВЦ с ненастроенной антенной и электронной (варикапной) перестройкой в диапазоне 250–2000 кГц при трёх видах связи. Настройка контура — потенциометр «НАСТРОЙКА» (управление варикапом); связь — кнопки C1, C2, C3 (C1 < C2 < C3) и M. Измеряемая величина: Uрез = (Uвых)max.
